# Lets Scan – QR Code Scanner

A simple and clean QR code scanner built using HTML, CSS, JavaScript, and the html5-qrcode library. This project uses your device camera to scan QR codes instantly.


## Features
• Scan any QR code using your camera<br>
• Works on both phone and laptop<br>
• Fast and lightweight<br>
• No backend required<br>

## Technologies Used
• HTML<br>
• CSS<br>
• JavaScript<br>
• html5-qrcode library<br>

## How to Use
1. Open the website ([venkatbhoogarbh.github.io/lets_scan](https://venkatbhoogarbh.github.io/lets_scan))<br>
2. Allow camera permission<br>
3. Point the camera at a QR code<br>
4. The scanned text/URL will appear on screen with the option to open it<br>
